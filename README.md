# flower-password-extension
花密秘密生成器edge浏览器插件

非原作者官方版本，根据原版仿制，加密算法与原版不同，请勿替代使用。

实现花密密码生成器的浏览器插件版本，包含以下功能：
- 基于记忆密码和网站域名生成唯一密码
- 支持多种密码长度和生成模式
- 自动填充密码到网页表单
- 本地存储记忆密码选项
- 包含完整的用户界面和功能逻辑

安装说明：
将flower-password-extension文件夹下载至电脑

打开浏览器管理扩展页

点开开发者选项

点击“加载解压缩的扩展”-选择flower-password-extension文件夹，点确定即可加载成功


在任意需要输入密码的页面，插件会根据该网站自动识别《区分代号》输入《记忆密码》后点一键填充即可输入生成后的密码。

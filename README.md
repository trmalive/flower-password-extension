# flower-password-extension
花密秘密生成器浏览器插件

实现花密密码生成器的浏览器插件版本，包含以下功能：
- 基于记忆密码和网站域名生成唯一密码
- 支持多种密码长度和生成模式
- 自动填充密码到网页表单
- 本地存储记忆密码选项
- 包含完整的用户界面和功能逻辑
